# AWS Cloud Practitioner Projects

A comprehensive collection of hands-on projects and labs for AWS Cloud Practitioner certification learning. This repository showcases practical experience with AWS services and cloud concepts.

## 📚 About This Repository

This repository documents my journey through the **AWS Cloud Practitioner certification** with real-world, hands-on projects. Each project demonstrates proficiency with key AWS services and cloud concepts.

**Target Audience**: Beginners learning AWS cloud computing  
**Certification**: AWS Certified Cloud Practitioner  
**Cloud Provider**: Amazon Web Services (AWS)

---

## 🚀 Projects Overview

### Beginner Level

#### 1. [EC2 Web Server](./01-ec2-web-server/)
- **Objective**: Launch and configure an EC2 instance to host a web server
- **Services**: EC2, Security Groups, Elastic IPs
- **Duration**: ~30 minutes
- **Cost**: ~$0.01/hour (t2.micro free tier eligible)
- **Status**: 📝 Planned

#### 2. [S3 Static Website](./02-s3-static-website/)
- **Objective**: Host a static website using S3 and CloudFront
- **Services**: S3, CloudFront, Route 53 (optional)
- **Duration**: ~45 minutes
- **Cost**: ~$0.50/month
- **Status**: 📝 Planned

#### 3. [RDS Database Setup](./03-rds-database/)
- **Objective**: Create and connect to a relational database instance
- **Services**: RDS, Security Groups, EC2
- **Duration**: ~60 minutes
- **Cost**: ~$0.20/day (free tier eligible)
- **Status**: 📝 Planned

#### 4. [Lambda Function & API Gateway](./04-lambda-function/)
- **Objective**: Build a serverless function triggered by API Gateway
- **Services**: Lambda, API Gateway, CloudWatch
- **Duration**: ~45 minutes
- **Cost**: ~$0.00/month (free tier eligible)
- **Status**: 📝 Planned

#### 5. [IAM Roles & Policies](./05-iam-roles-policies/)
- **Objective**: Create and manage IAM roles and permissions
- **Services**: IAM, CloudTrail
- **Duration**: ~30 minutes
- **Cost**: Free
- **Status**: 📝 Planned

#### 6. [VPC & Networking](./06-vpc-networking/)
- **Objective**: Design and configure a Virtual Private Cloud with subnets
- **Services**: VPC, Subnets, Internet Gateway, NAT Gateway
- **Duration**: ~60 minutes
- **Cost**: ~$0.05/month
- **Status**: 📝 Planned

### Intermediate Level

#### 7. [Auto Scaling & Load Balancing](./07-auto-scaling/)
- **Objective**: Set up auto-scaling groups and load balancers
- **Services**: Auto Scaling, ELB, CloudWatch
- **Duration**: ~90 minutes
- **Cost**: ~$0.50/day
- **Status**: 📝 Planned

#### 8. [CloudWatch Monitoring & Alarms](./08-cloudwatch-monitoring/)
- **Objective**: Monitor AWS resources and set up alarms
- **Services**: CloudWatch, SNS, CloudTrail
- **Duration**: ~45 minutes
- **Cost**: Free (basic monitoring)
- **Status**: 📝 Planned

#### 9. [SNS & SQS Messaging](./09-messaging-services/)
- **Objective**: Implement message queues and notifications
- **Services**: SNS, SQS, Lambda
- **Duration**: ~60 minutes
- **Cost**: ~$0.00/month (free tier eligible)
- **Status**: 📝 Planned
